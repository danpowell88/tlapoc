## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**As a** team, **I want** a shared component library and design tokens, **so that** web and apps look consistent and screens are faster to build.

A shared design system: colour/spacing/type tokens and the common components (buttons, inputs, cards, tables, modals, chips) used across the staff web and the apps. Establishes the visual language the prototype already demonstrates.

**Surface:** web · Flutter · **Touches:** component library, design tokens.

**Build steps (functional):**
- [ ] Design tokens (colour, spacing, type) defined once and shared
- [ ] A core set of reusable components built and documented
- [ ] Web and apps consume the same tokens/components
- [ ] A simple component gallery/reference exists

**References:** prototype visual language; PRD-01 (ADR-0005).
