## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**As an** owner, **I want** S4 fridge temperatures logged with alerts on excursions, **so that** out-of-range medicine is caught and quarantined before it is used.

Temperature logging for cold-chain storage (2-8C) with excursion detection and alerting. A breach raises an alert and flags affected stock as not-for-use pending review. Fed by the clinic's own wireless fridge monitor (and/or commercial loggers); charts show history.

**Surface:** web (staff) · API · hardware feed · **Touches:** temperature logs, excursion alerts, affected-stock flag.

**Build steps (functional):**
- [ ] Ingest temperature readings from the fridge monitor(s)
- [ ] Detect excursions outside 2-8C and raise an alert
- [ ] Flag affected stock as quarantined pending review
- [ ] Chart temperature history per fridge
