## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**As an** injector, **I want** to log an adverse event or complication linked to the treatment, product and lot, **so that** it is documented, the right emergency response runs, and a TGA report can be filed.

Capture an adverse event linked to the treatment/product/lot, classify seriousness, and target the correct DAEN dataset (medicine vs device). Includes the immediate complication-response path (e.g. vascular occlusion or anaphylaxis -> record hyaluronidase/adrenaline given -> raise routed jobs). A prefilled export supports the TGA report; the full reporting pathway lives in the Governance hub.

**Surface:** web (staff) · provider app · API · **Touches:** adverse event (seriousness, product, lot, DAEN target), complication response, jobs.

**Build steps (functional):**
- [ ] Log an adverse event linked to treatment, product and lot
- [ ] Classify seriousness and target the correct DAEN dataset
- [ ] Complication response: record emergency meds given and raise routed jobs
- [ ] Produce a prefilled export for the TGA report

**Prototype:** clinical-safety.png / gov-ae.png.
**References:** PRD-05 §6 (adverse events), REQ-CLIN-5, C12, AC5; routing -> PRD-08/11.
