## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**As** staff, **I want** scattered recall, needs-attention and unanswered-comms items merged into one queue, **so that** nothing falls through the cracks.

**Surface:** web (staff) · API · **Touches:** unified job queue, flagging, rules-based categorisation.

- [ ] Staff can flag any message into the queue
- [ ] Inbound comms auto-categorise into jobs by rules/keywords
- [ ] Ingest client-reported concerns
