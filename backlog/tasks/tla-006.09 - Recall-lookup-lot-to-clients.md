## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**As an** owner, **I want** to list every client who received a given lot, **so that** I can act immediately on a product recall or contamination concern.

Given a lot/batch number, instantly list every client and administration for that lot. The lookup reads the lot-to-client links written at administration. Recall execution (notifying clients, DAEN routing) is in the Governance hub epic; this is the lookup that powers it.

**Surface:** web (staff) · API · **Touches:** lot-to-client query.

**Build steps (functional):**
- [ ] Enter a lot number and list all clients/administrations for it
- [ ] Show administration details (date, practitioner, units)
- [ ] Export the list
- [ ] Hand off to recall execution (later epic)
