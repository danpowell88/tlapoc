## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**As an** owner, **I want** a one-click inspection-readiness pack and data-quality checks, **so that** I can face an audit confidently and keep the data clean.

A one-click pack assembling the compliance evidence an inspector expects, plus data-quality findings (active-but-unseen, completed-not-checked-in, duplicates, missing contacts, implausible dates) and an owner 'needs attention' exceptions digest.

**Surface:** web (owner) · API · **Touches:** evidence pack, data-quality findings, exceptions digest.

**Build steps (functional):**
- [ ] One-click inspection-readiness pack of compliance evidence
- [ ] Data-quality checks surface anomalies
- [ ] Owner 'needs attention' exceptions digest
