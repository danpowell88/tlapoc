## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**As a** designated RN prescriber, **I want** to prescribe S2-S4 in partnership with an authorised prescriber, **so that** the clinic can use the QLD endorsement (from Sept 2025) within its legal bounds.

A third prescriber identity in the prescription model: the designated RN prescriber, verified by AHPRA endorsement and tied to a recorded partnered authorised prescriber. The credential gate and the prescription.prescriber model must accept this role.

**Surface:** web (staff) · API · **Touches:** prescriber identity model, partnered-prescriber record, credential gate.

**Build steps (functional):**
- [ ] Accept the designated RN prescriber as a valid prescriber on a prescription
- [ ] Require a recorded partnered authorised prescriber
- [ ] Verify via AHPRA endorsement in the credential gate
- [ ] Reflect the role in scope checks (canInject / prescribing)

**References:** PRD-04 rev-5 (designated RN prescriber), requirements §6.1; pairs with PRD-01 RBAC.
