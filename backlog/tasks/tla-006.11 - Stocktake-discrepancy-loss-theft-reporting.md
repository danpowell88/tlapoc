## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**As an** owner, **I want** to run a stocktake and report discrepancies, **so that** S4 stock is regularly verified and any loss or theft is documented.

Stocktake against the ledger, discrepancy detection, and the ability to raise a loss/theft report from a discrepancy. Produces the records an auditor expects.

**Surface:** web (staff) · API · **Touches:** stocktake, discrepancy, loss/theft report.

**Build steps (functional):**
- [ ] Run a stocktake counting on-hand against the ledger
- [ ] Detect and record discrepancies
- [ ] Raise a loss/theft report from a discrepancy
- [ ] Retain stocktake history

**References:** PRD-04 §6, REQ-MED-10, C17, AC9.
