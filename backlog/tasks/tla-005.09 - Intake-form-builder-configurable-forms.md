## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**As an** owner/admin, **I want** to configure intake forms beyond the default, **so that** we can tailor questions per treatment type as our services grow.

A no-code builder to compose intake forms from question types, with validation rules, beyond the shipped default toxin intake. An enhancement over the MVP default form.

**Surface:** web (staff/admin) · API · **Touches:** form definitions, question types, validation.

**Build steps (functional):**
- [ ] Compose forms from question types with ordering
- [ ] Set validation/required rules per question
- [ ] Assign forms to treatment types
- [ ] Versioned like consent templates
