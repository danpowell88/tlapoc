## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**As a** compliance officer, **I want** a guided workflow when a breach is suspected, **so that** we assess eligibility and notify correctly within our obligations.

Breach handling: flag a suspected breach, run an assessment of whether it is notifiable, and — if eligible — produce regulator (OAIC) and individual notifications, all recorded in a breach register.

**Surface:** web (staff) · API · **Touches:** breach assessment, register, notifications.

**Build steps (functional):**
- [ ] Flag a suspected breach to start an assessment
- [ ] Assess whether the breach is eligible/notifiable
- [ ] Generate regulator and individual notifications when eligible
- [ ] Record everything in a breach register

**References:** PRD-01 §6/§8 (breach), C22, AC7.
<!-- SECTION:DESCRIPTION:END -->
